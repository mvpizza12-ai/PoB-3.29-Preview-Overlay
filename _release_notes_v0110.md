## English

Unofficial **file overlay** for [Path of Building Community](https://github.com/PathOfBuildingCommunity/PathOfBuilding) approximating Path of Exile **3.29 (Curse of the Allflame)** before official PoB data lands.

**Temporary.** When official PoB updates for 3.29, remove this overlay.  
Not affiliated with GGG or official Path of Building Community.

### Install
1. Download and extract the zip below
2. Fully quit Path of Building (tray too)
3. Double-click **Install.bat** → press `Y`
4. Launch PoB → press **F5**
5. Items tab should show `Vestigial...` / `Foulborn...` / `Volatile Vaal...`

### Warnings
- Sharing a **pobb.in / build code alone does not include this overlay** — the other person must install the zip too.
- See FEATURES.md for curated vs patch-based details.
